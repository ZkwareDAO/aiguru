### 题目分析

题目已知条件为：在$\triangle ABC$中，内角$A,B,C$所对的边分别为$a,b,c$，已知$A=\frac{\pi}{4}$，$b^2-a^2=\frac{c^2}{2}$。

### 解答过程

#### （1）求$\tan C$的值

首先，利用余弦定理和正弦定理来构建关系。

已知$A=\frac{\pi}{4}$，则$\sin A = \cos A = \frac{\sqrt{2}}{2}$。

由余弦定理得：$a^2 = b^2 + c^2 - 2bc\cos A$，代入$\cos A = \frac{\sqrt{2}}{2}$，得$a^2 = b^2 + c^2 - \sqrt{2}bc$。

又给定$b^2 - a^2 = \frac{c^2}{2}$，将$a^2 = b^2 + c^2 - \sqrt{2}bc$代入，得$b^2 - (b^2 + c^2 - \sqrt{2}bc) = \frac{c^2}{2}$，化简得$c^2 - \sqrt{2}bc = -\frac{c^2}{2}$，即$c^2 + \frac{c^2}{2} = \sqrt{2}bc$，得到$c = \sqrt{2}b$。

由正弦定理$\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C}$，代入$c = \sqrt{2}b$和$a^2 = b^2 + c^2 - \sqrt{2}bc$得到的$a$的表达式，可以解出$\sin C$和$\cos C$，进而求得$\tan C = \frac{\sin C}{\cos C}$。

#### （2）若$\triangle ABC$的面积为7，求$b$的值。

三角形面积公式为$S = \frac{1}{2}bc\sin A$，代入$S = 7$，$c = \sqrt{2}b$和$\sin A = \frac{\sqrt{2}}{2}$，得到$7 = \frac{1}{2}b\sqrt{2}b\frac{\sqrt{2}}{2}$，解出$b$的值。

### 检查学生答案

学生提供的答案是$(10.99/11.79)$，这似乎是一个计算结果，但没有给出具体的解题过程。根据上述分析，我们可以看出，学生可能直接计算了$b$的值，但没有展示如何求解$\tan C$的完整过程。因此，学生答案缺乏对问题（1）的解答，且没有展示计算过程，即使计算结果可能正确（需具体计算验证），也不能完全确定其正确性。

### 结论

为了批改答案，我们需要看到学生的完整解答过程，特别是对于问题（1）的解答。如果学生只是给出了一个数值结果，而没有展示如何得到这个结果的过程，那么这个答案是不完整的。教师应要求学生详细展示解题步骤，以确保其理解了求解方法。
