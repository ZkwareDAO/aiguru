学生解答的这道题目涉及三角恒等变换和正弦、余弦定理的综合应用。让我们逐一检查步骤和结论的正确性。

### 第一部分
题目要求利用余弦定理求解，具体步骤如下：

1. **余弦定理应用**：学生首先正确地应用了余弦定理来表示cosA，即\(\cos A = \frac{b^2 + c^2 - a^2}{2bc}\)，并得出\(\cos A = \frac{\sqrt{2}}{2}\)。这一步是正确的，因为根据题设条件，A角的余弦值确实为\(\frac{\sqrt{2}}{2}\)，意味着A角是45度。

2. **正弦定理与三角恒等变换**：学生接下来利用正弦定理和三角恒等变换推导sinB与sinC的关系。在这一过程中，学生正确使用了sin^2A + cos^2A = 1等恒等关系，以及正弦和余弦的二倍角公式。最终得出\(\tan C = \frac{\sqrt{2}}{2}\)。这一结论也是正确的，因为根据A角为45度，且已知cosA和sinA的值，可以推导出C角的正切值。

### 第二部分
题目要求计算三角形的面积以及边长b的值：

1. **计算sinC和cosC**：学生计算出\(\sin C = \frac{\sqrt{2}}{3}\)，\(\cos C = \frac{\sqrt{7}}{3}\)，这一步是正确的，与第一部分中得出的tanC值相吻合。

2. **计算sinB**：利用sinB = sin(A+C)的公式，学生正确地计算出了\(\sin B = \frac{3\sqrt{14}}{6}\)。这一计算过程和结果都是正确的。

3. **计算三角形面积和边长b**：学生利用面积公式\(\Delta ABC = \frac{1}{2}bc\sin A = 3\)，正确地计算出了\(bc = 6\sqrt{2}\)。然后，利用余弦定理\(a^2 = b^2 + c^2 - 2bc\cos A\)，最终得出\(b = 2\sqrt{2}\)。这一系列计算步骤和结果都是正确的。

### 总结
学生对这道题目的解答是正确的，无论是三角恒等变换的应用，还是正弦、余弦定理的使用，都表现出了良好的理解和掌握。计算过程严谨，结论准确。如果要进一步提高，可以建议学生在解题过程中更加注意步骤的清晰度和逻辑的连贯性，使解题过程更加易于理解和跟踪。
